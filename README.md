# Access Heart - Virtual Cardiology for Rural America

Premium healthcare landing page for accessheart.com.

## Tech Stack
- React, TypeScript, Tailwind CSS, Framer Motion, Shadcn UI
- Express.js backend, Vite build
- Design: Modern Clinical Warmth (coral red + navy + amber)

## Setup
```
npm install
npm start
```

## Key Files
- client/src/pages/Home.tsx - Main landing page
- client/src/components/sections/ - All section components
- client/src/components/layout/ - Navbar and Footer
- client/src/index.css - Styling
- client/src/pages/EmailPreview.tsx - Partnership email template
- attached_assets/ - Images and assets
